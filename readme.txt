 اجرای سرور MLflow
MLflow برای ذخیره و نمایش اطلاعات به یک سرور نیاز دارد.
برای راه‌اندازی سرور روی لوکال، در ترمینال دستور زیر را اجرا کن:

bash
Copy
Edit
mlflow ui
این دستور پنل وب MLflow را روی آدرس http://127.0.0.1:5000 اجرا می‌کند.

می‌توانی با مرورگر به این آدرس بروی و نتایج آزمایش‌ها، مدل‌ها، پارامترها و متریک‌ها را ببینی.
